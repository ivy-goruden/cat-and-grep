# s21_cat & s21_grep

Реализация базовых утилит `cat` и `grep` на языке C.

## Структура проекта

- `cat/`: Исходный код и тесты для `s21_cat`.
- `grep/`: Исходный код и тесты для `s21_grep`.

## Сборка

Для сборки утилит перейдите в соответствующую директорию и выполните `make`:

```bash
# Сборка s21_cat
cd cat
make

# Сборка s21_grep
cd ../grep
make
```

## s21_cat

`s21_cat` — утилита для конкатенации и вывода файлов.

### Использование

```bash
./s21_cat [ФЛАГИ] [ФАЙЛ]...
```

### Флаги

| Флаг | Описание |
|------|----------|
| `-b` (GNU: `--number-nonblank`) | Нумерует только непустые строки (перекрывает `-n`). |
| `-e` | Отображает непечатные символы (как `-v`) и символ ` в конце каждой строки. |
| `-E` | Отображает символ ` в конце каждой строки. |
| `-n` (GNU: `--number`) | Нумерует все выходные строки. |
| `-s` (GNU: `--squeeze-blank`) | Сжимает несколько смежных пустых строк в одну. |
| `-t` | Отображает непечатные символы (как `-v`) и табы как `^I`. |
| `-T` | Отображает табы как `^I`. |

## s21_grep

`s21_grep` — утилита для поиска строк, соответствующих регулярному выражению, в текстовых файлах.

### Использование

```bash
./s21_grep [ФЛАГИ] [ШАБЛОН] [ФАЙЛ]...
```

### Флаги

| Флаг | Описание |
|------|----------|
| `-e` | Шаблон. |
| `-i` | Игнорирует различия регистра. |
| `-v` | Инвертирует смысл поиска (выводит несоответствующие строки). |
| `-c` | Выводит только количество совпадающих строк. |
| `-l` | Выводит только имена файлов с совпадениями. |
| `-n` | Предваряет вывод номером строки. |
| `-h` | Не выводит имена файлов в результатах. |
| `-s` | Подавляет сообщения об ошибках о несуществующих или недоступных файлах. |
| `-f` | Получает шаблоны из файла. |

## Тестирование

Тесты находятся в директории `test/` каждой утилиты.

```bash
# Запуск тестов s21_cat
cd cat/test
./s21_cat_test.sh

# Запуск тестов s21_grep
cd grep/test
./s21_grep_test.sh
```
